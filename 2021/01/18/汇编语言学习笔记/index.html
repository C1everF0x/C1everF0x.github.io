0X00 前置基础知识

安装汇编工具dosbox 和 二进制大佬常用编辑器vim ，简单使用

关于doxbox：本质是一个模拟32位操作系统的模拟器

简单的编译、链接了一个文件

  • CPU（中央处理器）是将机器指令（一串二进制数字）转换为高低电平来驱动计算机运行的一个硬件

  • 什么是机器指令？

    0101010 ，形如此的二进制代码

  • 机器指令由谁执行？

    CPU ，CPU 将其翻译成高低电平来驱动计算机运行

  • 机器指令和汇编指令有什么关系？

    通过编译器，将汇编指令翻译成机器指令

  • 什么是编译器？

    通俗讲就是“翻译”软件，用来将高级一点的语言翻译成低级一点的语言，比如从汇编指令翻译成机器指令

    编译器在程序可运行前有几个步骤要做：
    读取你给它的源代码中的独立字符。
    将字符分类为字，数字，符号和操作符。
    获取已排序完的字符，并通过将它们与模式匹配相匹配和生成操作树来确定它们尝试进行的操作。
    迭代上一步中生成操作树中的每一个操作，并生成等效的二进制文件。
    当编译器立刻从一个操作树转化到二进制文件时，它实际生成了汇编代码，汇编代码随后被汇编/编译成二进制文件。汇编像是更高级的，人类可读的二进制文件。

  • 什么是伪指令？

    告诉编译器，每个地方该如何进行“翻译”

  • 机器指令（汇编指令）存放在哪里？

    CPU 中存放了一部分指令和数据，其他绝大多数指令和数据都是存放在内存中，而其中又绝大部分存放在内存条中，指令和数据在内存中是没有任何区别的

  • 机器指令（汇编指令）是以什么形式存放在内存中的？

    二进制数据，010101010，写成十六进制是方便阅读

  • 内存（存储单元）的编号怎么算？

    0123456789 ABCDEF

  • 内存的计量单位？

    等价 意义
    1 byte = 8 bit B = bit
    1 KB = 1024 byte K = kilo
    1 MB = 1024 KB M = million
    1 GB = 1024 MB G = giga

  • 内存编号为什么是从 0 开始的？

    CPU 和内存条之间联系需要通过电路（导线），其中有三根线分别是地址线、数据线和控制线

    内存编号信息 地址线（内存地址）
    数据信息 数据线
    读写信息 控制线

    由于物理上的限制，电路只能表示 0 或 1

  • 什么叫寻址能力？

    寻址能力是指计算机（CPU）寻找内存大小的能力，由地址线决定，地址线的数量决定了能传多少个 0 和 1，多少个 0 和 1 则对应着内存编号信息（内存地址）

  • 数据线

    决定了 CPU 的和其他部件进行数据传输时一次性能够传输多少数据的能力，比如有 8 根数据线，则一次性可以传输 8 位 bit 也就是一个字节的数据，16 根数据线就是一次传输两个字节的数据，可以理解为一根数据线能传输 1 bit 数据

  • 控制线

    决定了 CPU 对其他部件进行控制的能力

  • CPU 和计算机各个部件之间的关系？

    CPU 将内存条、显存、ROM、RAM内存都看成内存空间，给内存空间分段分类分发编号（虚拟内存地址）

  • 什么是 RAM 内存？

    全称Random-Access Memory(随机存取存储)，叫主存、内存也叫运行内存，允许读取和写入，断电后内存中指令和数据丢失

  • 什么是 ROM 内存？

    全称Read Only Memory(只读存储)，也就是硬盘，只允许读取，断电后内存中指令和数据还存在，一般用于启动计算机上

0X01 寄存器与寄存器内存访问

寄存器

用的《汇编语言第四版》——王爽，讲的是16位的8086处理器，寄存器是十六位的

  • 16位通用寄存器四个，每个可分为两个 8 位寄存器，高八位和低八位的
AX AH + AL
BX BH + BL
CX CH + CL
DX DH + DL
  • 内存的最小单元是字节，8 bit，CPU从内存中读取一个字节，8 位数据存入 8 位寄存器中。16 根数据线的宽度决定了 CPU 一次性能够读取 2 字节（1 字）的数据
  • 数据和寄存器之间要保证一致性，8 位寄存器给 8 位寄存器，8 位数据给8 位数据，16 位寄存器给 16 位寄存器，16 位数据给16 位数据
