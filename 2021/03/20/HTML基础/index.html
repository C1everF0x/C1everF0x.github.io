随便学学

元素

  • HTML 元素以开始标签起始，结束标签终止
  • 元素内容是开始标签和结束标签中间的内容
  • HTML 元素可以嵌套包含其他 HTML 元素
  • HTML 大部分元素被定义为块级元素或者内联元素
  • 内联元素在显示时候不另起一行，如<b>、<td>、<img>
  • 块级元素在显示的时候会另起一行，如<h1>、<p>、<div>

属性

  • HTML 元素可以设置属性
  • 属性一般附加在开始标签
  • 属性以名称/值对的形式出现

  • 常用属性

    属性 描述
    class 为 HTML 元素定义一个或多个类名
    id 定义元素的唯一 id
    style 规定元素的行内样式
    title 元素的额外信息

  • HTML 标准属性参考手册

链接

添加 target 属性可以让链接在新窗口打开

id 属性可以在 HTML 文档中做书签标记

点击 访问 tips就会定位到一个提示所在的地方

图像

图像的名称和尺寸以属性的形式提供，src属性表示图像的 url 地址，alt属性为替代文本，浏览器显示不出图片的时候显示替代文本，width和height定义图片的宽和高

表格

<table>标签定义一个表格，<tr>标签每次定义一行，<td>标签定义一个单元格，<th>标签定义表头，会将表头内容显示为粗体居中， border属性设置表格是否有边框，还有很多属性…

列表

  • 无序列表，<ul>标签开始，<li>标签定义内容

    浏览器显示

    • C1everF0x
    • Aimerl0

  • 有序列表，<ol>标签开始，<li>标签定义内容

    浏览器显示：

    1. C1everF0x
    2. Aimerl0

  • 自定义列表，<dl>标签自定义列表，<dt>标签自定义列表项，<dd>标签自定义列表项的描述

    浏览器显示：

    C1everF0x

    - My latest id

    Aimerl0

    - My favourite id

区块、布局

<div>块级元素，没有实际意义，用于文档布局

<span>内联元素，同样没有实际意义，用于做文本的容器

表单和输入

<form>标签用来表示一个包含表单元素的区域，<input>标签用来输入，输入类型由type属性来定义

  • text属性定义输入一段文本，默认 20 字符
  • password属性定义密码字段，不会明文显示
  • radio属性定义单选框
  • checkbox属性定义多选框
  • action属性定义内容传输目的文件文件名，method属性定义传输方式
  • submit属性定义提交

脚本

<script>标签定义客户端脚本，可以直接包含脚本语句，也可以通过src属性指向外部脚本文件

noscript标签用于写浏览器不支持脚本时候的替换语言

CSS

三种样式添加到 HTML 中，CSS 层叠样式表，用于渲染 HTML 文档中的元素

在样式属性中可添加任何 CSS 属性，比如：

定义背景颜色：background-color:

字体：font-family

颜色：color:red（红色）

字体大小：font-size:20px

文本对齐：text-align:center（居中对齐）

  • 内联样式：在 HTML 元素中使用style属性

  • 内部样式表：在 HTML 文档头部<head>标签内使用<style>标签调用 CSS

    在<head>标签内通过<style>标签来定义内部样式表

  • 外部引用：使用外部 CSS 文件

    通过更改文件来改变整个站点的外观，<link>标签用于定义资源的引用位置
