不知道 Web 的每日一题还能坚持多久，做到以前那种大比赛的 Web 题时发现一天时间很难吃透，或者根本就是一天时间内搞不懂，还听协会里的 pwn 师傅说一道 pwn 题能打三四天，一个星期，所以可能以后每日一题往各个方向的基础题发展，下一步准备把固态买回来重装一遍系统，然后进军 pwn！！！

[极客大挑战 2019]BuyFlag

进入 pay.php查看源码发现 waf ，同时要求 money = 1000000000

in_numeric()函数对于第一个空格字符会条跳过，接着后面的判断，绕过方式就是 password=404%20，但是传参 money = 100000000 时发现密码正确数字过长

用科学计数法 money = 1e10可以绕过，拿到 flag

值得注意的点，Cookie: user = 0 要改成 Cookie: user = 1，可能对应题目的话

百度wp发现大师傅们都是直接猜测 money 的比较函数是 strcmp ，直接用数组绕过
