写在前面

之前刷 BUU 的时候做了一些2019极客大挑战的题，感觉挺好的，适合练手复习考点

今年从协会扫地学长那得知又开赛了，得整一整

当我得知这是成都信息工程大学的招新赛时，这 CTF 氛围实名羡慕了。。。

Welcome

提示：In addition to the GET request method, there is another common request method…

进去以后状态码405，结合提示 burp 抓包改 POST 方法，发现有 PHP 代码

常规的 sha1数组绕过，传参roam1[]=123&roam2[]=abc来到 phpinfo 页面，题目要在 phpinfo 界面收集信息，仔细观察发现敏感信息

进入 f1444aagggg.php 在响应头处拿到 flag

朋友的学妹

进去要我试一试查看源代码，妹子还挺好看的，但我是个无情的做题机器

源代码里有 flag 的 base64 编码，解码以后得 flag：SYC{F1@_4s_h4Lpfullllll}

EZwww

提示：备份是个“好🤺”习惯

尝试www.zip，下载下来了备份，flag!!!!.txt照例是一个假的 flag ，审index.php

要求 POST 方法传参一个变量a要求等于base64_decode('c3ljbDB2ZXI=')，即sycl0ver，所以传个参就好

EZgit

.git泄露经典考点了，去github上下载工具GitHack，开锤

GitHack 只能python2运行

工具跑完打开fl4g.php显示 flag is toooo old!，所以需要用git拉回上一个版本

输入财富密码git reset --hard 3796466675a1db323e42170def92bee71344a2ee，再打开fl4g.php拿到 flag
